Predmet: Softver inženjering

Opis projekta: 
Ovaj projekat predstavlja jednostavnu web aplikaciju za upravljanje korisnicima koristeći PHP i MySQL bazu podataka. 
Aplikacija koristi CRUD funkcionalnosti koje omogućavaju dodavanje, pregled, izmjenu i brisanje korisnika iz baze podataka (Create, Read, Update, Delete).

U projektu su upotrijebljenji:
PHP
MySQL
HTML
CSS
JavaScript
Bootstrap

Funkcionalnost aplikacije:
Dodavanje novih korisnika
Prikaz svih korisnika u tabeli
Izmjena postojećih korisnika
Brisanje korisnika
Pretraga korisnika
